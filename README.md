# Задача : разработать систему показа модальных окон.

## Технические требования:

- Код должен быть написан на TypeScript с полной типизацией
- В качестве UI-фреймворка использовать Vue 3 с Single-File Components (SFC) и Composition API
- Запрещено использовать готовые библиотеки компонентов - все необходимые компоненты нужно реализовывать самостоятельно

## Состав задачи:
### 1. Создать компонент кнопки

Кнопка должна уметь нажиматься и ей должно быть можно задать текст
Внешний вид, остальные возможности - на усмотрение кандидата.

### 2. Создать компонент модального окна:

Компонент должен состоять из:
- title
- body (любая верстка без ограничений)
- footer (набор кнопок действий) 

Должна быть возможность закрыть модальное окно:
- крестиком справа сверху
- нажатием вне модального окна
- нажатием Esc

Нужно предусмотреть scroll:
- внутри модального окна, если body не помещается
- всего модального окна, если размер окна браузера меньше, чем размер модального окна

### 3. Спроектировать и реализовать систему показа модальных окон

Система должна поддерживать:
- императивный интерфейс показа и скрытия, т.е. её должно быть удобно использовать например для обработки нажатия кнопки:
```
	<button @click="show(/* params */)>click me<button>
```
- показ нескольких модальных окон: открытие очередного окна временно скрывает предыдущее (и так далее), закрытие текущего окна вызывает показ предыдущего (если очередь показа не пуста)

Будет бонусом, если будет реализована система управления фокусом:
- при открытии модального окна фокус переходит в него и устанавливается на default focusable элемент
- при закрытии не модального окна фокус возвращается на предыдущее модальное окно / страницу, в то место где был

### 4. Создать тестовую страницу для демонстрации работы

Минимальная функциональность страницы:
- кнопка, открывающая самый простой вид модального окна
- кнопка, открывающая модальное окно со скроллом контента внутри
- кнопка, открывающая высокое модальное окно, вызывающее scroll на уровне страницы
- кнопка, открывающая модальное окно с заголовком "Окно №1" и кнопками: "Открыть ещё", "Закрыть это окно", "Закрыть все окна". "Открыть ещё" открывает ещё одно модальное окно, с заголовком "Окно №{n+1}" и такими же кнопками

Можно добавить другие примеры на усмотрение кандидата.
